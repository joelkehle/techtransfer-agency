Inteum Access

Read and write access by agent role

Inteum is UCLA TDG's IP management database. The IP Agency reads from and writes to Inteum to eliminate manual data entry, ensure agents work from authoritative records, and store evaluation outputs alongside the cases that generated them.

The table below maps each agent role to the Inteum record types it needs to access. Today, no integration exists — all disclosures are uploaded manually. Phase 1 asks for read-only access to Invention Disclosures only.

What we're asking for today (Phase 1): Read-only access to Invention Disclosures. This lets the system pull a disclosure by case number instead of requiring manual PDF upload. Non-destructive. No writes. Everything else in this table is future state.
